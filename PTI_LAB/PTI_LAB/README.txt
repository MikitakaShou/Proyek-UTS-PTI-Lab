## GAMBAR KARAKTER
  - 4 karakter makanan yang tersedia (tahu bulat, nanas, sushi, burger)
  - untuk merubah karakter, harus membeli dengan money.

## CARA BERMAIN
- Membuka website / file index.html
- Klik karakter untuk menghasilkan uang
- belilah autoclicker sebanyak-banyaknya untuk menghasilkan lebih banyak uang dan mempermudah game
- Jika uang cukup, lakukanlah perubahan karakter sesuai keinginan
- Objektif dari permainan ini adalah mengumpulkan uang sebanyak-banyaknya


## PERATURAN GAME
Terdapat 5 button yang tersedia:

3 button diatas:
1) burger 
2) sushi
3) pineapple
fungsinya untuk mengganti karakter ketika uang cukup

2 button dibawah:
1) autoclicker : jika sudah membeli autoclicker,maka game akan mengclick secara otomatis dan uang akan bertambah dengan sendirinya,dan juga akan menambah jumlah money per click yang dihasilkan.
2) Buka Cabang : Jika uang cukup maka game akan langsung membuka cabang baru dengan berpindah html dan jika tidak cukup maka ada alert dan cabang tidak terbuka

3 Dan juga user dapat memilih makanan secara acak tidak perlu sesuai urutan
contoh:
1)user dapat memilih burger lalu refresh dan memulai dengan sushi begitu juga dengan pineapple
2)user dapat memilih burger lalu ke pineapple ataupun sushi dan sebaliknya

4 Kami hanya menyediakan 1 tempat utama dan 1 cabang jadi di tampilan cabang tidak ada button untuk membuka cabang lagi.

5 Kami membuat semua harga $100 agar ketika diperiksa akan memudahkan penilai untuk menilai proyek kami.
